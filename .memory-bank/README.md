# Memory Bank — ChatCenter

Это единственный источник истины для проекта ЦентрЧатов / ChatCenter.

## Структура файлов

| Файл | Содержимое | Читать когда |
|------|------------|--------------|
| `README.md` | Этот файл — навигация по Memory Bank | Всегда |
| `architecture.md` | Архитектура, слои, схема процессов, структура папок | Перед любой задачей |
| `coding-rules.md` | Стиль кода, IPC, WebView, безопасность | Перед написанием кода |
| `workflow.md` | Правила работы AI, планирование, чеклист | Перед началом задачи |
| `common-mistakes.md` | Типичные ошибки и как их избежать | При ошибках/отладке |
| `features.md` | Реализованные функции, статус, Changelog | Перед добавлением функций |
| `decisions.md` | Ключевые архитектурные решения (ADR) | При принятии решений |
| `api.md` | IPC-каналы, форматы сообщений, DTO | При работе с IPC |
| `messengers.md` | Интеграция мессенджеров: селекторы DOM, особенности | При работе с WebView |
| `ai-integration.md` | Провайдеры ИИ, промпты, форматы запросов | При работе с ИИ |
| `autoreply.md` | Логика авто-ответчика, схема правил, расписание | При работе с авто-ответом |

## Как обновлять Memory Bank

- Обновляй файл сразу после решения задачи, пока контекст свежий
- Не дублируй информацию — ссылайся на другой файл если нужно
- Устаревшую информацию удаляй или помечай как `[DEPRECATED]`
- Добавляй дату к важным решениям: `(3 марта 2026)`

## Соглашения

- Версии решений пишем как: `ADR-001`, `ADR-002` и т.д.
- Статусы: ✅ Сделано | 🚧 В работе | ❌ Отменено | 📋 Запланировано
- Мессенджеры кодируем: `TG` (Telegram), `WA` (WhatsApp), `VK` (ВКонтакте), `VB` (Viber)
